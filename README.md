Веб‑додаток для управління автосервісом (СТО), створений на Flask + MySQL. Підтримує ролі **адмін**, **механік**, **клієнт**, ведення замовлень, призначення механіків, використання запчастин та роботу з базою даних.

---

## 📌 Основні можливості

### 👑 **Адмін**

* Додає та керує механіками
* Створює клієнтів
* Додає автомобілі
* Створює та призначає замовлення механікам
* Переглядає всі замовлення
* Має повний контроль над системою

### 🔧 **Механік**

* Бачить лише призначені йому замовлення
* Додає коментарі до замовлення
* Обирає використані запчастини зі складу
* Змінює статус роботи

### 👤 **Клієнт**

* Бачить лише свої автомобілі
* Переглядає свої замовлення та статуси

---

## 🗄️ Стек технологій

* **Python 3.10+**
* **Flask**
* **MySQL / MariaDB**
* **SQLAlchemy ORM**
* **Werkzeug** (хешування паролів)
* **dotenv** для конфігурації

## ⚙️ Налаштування .env

Створіть файл `.env` у корені:

```
DB_USER=root
DB_PASS=your_password
DB_HOST=localhost
DB_NAME=sto_test
SECRET_KEY=your_secret_key
```

---

## 📥 Встановлення

### 1. Клонування репозиторію

```
git clone your_repo_url
cd sto_project
```

### 2. Встановлення залежностей

```
pip install -r requirements.txt
```

### 3. Створення бази даних

```
mysql -u root -p < dump.sql
```

### 4. Створення адміністратора

```
python create_admin.py
```

### 5. Запуск сервера

```
python app.py
```
